Presentación

En la actualidad los datos y la información tienen una presencia constante en nuestras vidas. Tradicionalmente la capacidad de extraer información de los datos fue aprovechada por las corporaciones para tomar decisiones operativas y comerciales. Contar con datos y tener las capacidades para extraer información y tomar decisiones es una fuente de poder.

Los datos y la información pueden ser también utilizados para mejorar la vida de todas las personas, y para ello deben ser insumos para la implementación, monitoreo y evaluación de las políticas públicas. Producir datos de calidad y utilizarlos para gestionar políticas públicas son pasos fundamentales para mejorar la vida de las personas.

Para esto es necesario planificar la obtención de datos y el monitoreo y evaluación de sus resultados, para evaluar el impacto que tiene la política pública en la sociedad. Esto implica un cambio cultural sobre el modo en que se toman decisiones y se analizan las políticas públicas. Se trata, de este modo, de pasar de una visión puramente cualitativa del diseño de políticas a una visión mixta, que también involucra el análisis cuantitativo.

Sin embargo, no cualquier utilización o evaluación de datos responde a una visión igualitaria y democrática de la política pública. Podría suceder que sólo se garantice la eficiencia dejando de lado implicancias de justicia e igualdad. Por ello, la perspectiva de derechos humanos debe ser transversal al análisis de los procedimientos. No es solo una mayor eficiencia lo que debe buscar una perspectiva de datos en las políticas públicas. Asimismo, las políticas y usos de datos deben garantizar la protección de datos personales y el derecho a la privacidad.

En esta materia revisaremos qué implica utilizar una metodología cuantitativa o empírica para la determinación e implementación de las políticas públicas. Analizaremos las discusiones teóricas y prácticas sobre la toma de decisiones basadas en evidencias, y la utilidad que pueden tener los datos. Pensaremos cuáles son las consecuencias que la utilización masiva de datos puede tener sobre excluidos sociales o disidentes políticos.

Luego, pasaremos a una parte más práctica, sobre cómo realizar análisis desde los datos públicos, tanto de la EPH, como del Censo como de otras bases de datos públicas, que nos permitirán evaluar las políticas públicas de los últimos años. Para ello, esta materia aprenderemos diversas herramientas de la ciencia de datos para llevar adelante el análisis empírico. Para ello, deberemos aprender a utilizar software de análisis estadístico, y conoceremos el tipo de operaciones que puede hacerse para analizar y procesar datos. Con ayuda de la Inteligencia Artificial, llevaremos adelante estos análisis, por nuestra propia cuenta.

Objetivos

El objetivo del curso es brindar herramientas metodológicas para poder planificar y evaluar una política pública basada en datos cuantitativos. Se espera que las y los alumnos puedan identificar las diversas metodologías de las ciencias sociales, cualitativas y cuantitativas, y puedan decidir cuál conviene utilizar en cada caso en la realización de una planificación, gestión y monitoreo de una política pública.

Al mismo tiempo, se espera que las y los alumnos cuenten con nociones básicas de la ciencia de datos, para analizar, procesar y graficar datos para la realización de informes.

Metodología

En el curso se brindarán herramientas teóricas, y luego las utilizaremos de modo práctico. Para ello, además de las lecturas de la teoría, utilizaremos software para la ciencia de datos y aprenderemos a utilizarlo. Comenzamos con una introducción a la ciencia de datos, utilizando ejemplos jurídicos sencillos para familiarizarnos con la herramienta.

Se espera que antes del comienzo del curso las y los estudiantes hayan instalado en sus computadoras el software de fuente abierta R (https://cran.r-project.org/), el entorno RStudio para utilizarlo (https://posit.co/products/open-source/rstudio/), y utilizaremos un manual para utilizarlo (https://cdaj.netlify.app/). Pueden conseguir un tutorial para la instalación aquí.

En una última etapa, juntaremos ambos saberes — la parte teórica sobre la toma de decisiones basadas en evidencia y la utilización de la Ciencia de Datos — para evaluar por nuestra cuenta el impacto del programa IFE implementado durante el 2020 en Argentina.
